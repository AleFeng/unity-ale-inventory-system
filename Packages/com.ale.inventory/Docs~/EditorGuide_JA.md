## エントリ一覧の共通操作（すべてのシステムの中央カラム）

6 大システムタブの中央カラムの「エントリ一覧」（アイテム / 倉庫 / ショップ / ブループリント / 装備グループ / スキル）は構造が統一され、操作も一貫しています：

- **2 行構造**：各エントリは「列名ヘッダー行 + 値行」の 2 行を占め、行の高さは一致します。ヘッダーは各列のフィールド名を表示 —— ID の後に順に**名称 / 説明**（エントリの `displayNameText` / `descriptionText` のプレーンテキストフォールバックを読む）、次に各システム固有の列（種類 / 主グループ / スロットグループ…）。
- **テンプレートフィルタ + 検索**：上部のテンプレートフィルタタグバーと検索ボックスで一覧の範囲を絞ります。
- **ドラッグ並べ替え**：左の `≡` ハンドルを長押ししてドラッグし、エントリ順序を調整します。
- **上下キーで選択切り替え**：あるエントリを選択後、`↑` / `↓` で現在可視（フィルタ済み）のエントリ間を 1 行ずつ選択切り替え。新しい選択項目が可視領域外のとき、一覧は自動的に 1 行スクロールして視野に戻します。検索ボックス / 右側 Inspector のテキストボックスを編集中は方向キーを奪いません。
- **追加 / 削除**：「テンプレートから追加」/「クイック追加」でエントリを追加、行末の `✕` で削除します。
