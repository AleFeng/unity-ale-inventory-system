## 条目列表通用操作（所有系统的中间列）

六大系统页签中间列的「条目列表」（道具 / 仓库 / 商店 / 蓝图 / 装备组 / 技能）结构统一，操作一致：

- **两行结构**：每个条目占「列名表头行 + 值行」两行，行高一致；表头显示各列字段名——ID 之后依次显示**名称 / 描述**（读条目的 `displayNameText` / `descriptionText` 纯文本 fallback），再是各系统特有列（类型 / 主分组 / 槽组…）。
- **模板过滤 + 搜索**：顶部模板过滤标签栏与搜索框缩小列表范围。
- **拖拽重排**：左侧 `≡` 句柄长按拖动调整条目顺序。
- **键盘上下键切换选中**：选中某条目后，按 `↑` / `↓` 在当前可见（已过滤）条目间逐行切换选中；新选中项超出可视区时列表自动滚动一行将其带回视野。正在编辑搜索框 / 右侧 Inspector 文本框时不劫持方向键。
- **添加 / 删除**：「从模板添加」/「快速添加」新增条目；行尾 `✕` 删除。
